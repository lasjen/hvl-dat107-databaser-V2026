# Eksempel 01 - XSD

## Fil: birken-paamelding.xml
Denne filen inneholder påmeldingsdata for et birken arrangement.

## Fil: birken-paamelding.xsd
XML Schema Definition (XSD) for validering av birken påmeldingsdata.
Alle definisjoner er bygd opp hierarkisk.

## Fil: birken-paamelding-2.xsd
En alternativ XSD der SimpleType og ComplexType er definert først i skjemaet.

## Fil: validate.sh
Bash script for å validere birken-paamelding.xml mot birken-paamelding.xsd ved hjelp av xmllint.
